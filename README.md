# Parallel libdeflate compression with paraz

## Dependencies
- [libdeflate](https://github.com/ebiggers/libdeflate).
- OpenMP.

## Compilation
```
mkdir build
cd build
cmake ..
make -j
```
This will create the paraz executable in the `build/bin` directory.
